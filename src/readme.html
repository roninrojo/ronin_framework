Para usar el framework

En la raíz de este proyecto:

git archive master | tar -x -C ~/git/static/static/promo-test/landing/dev/nombre_de_la_landing

o, si no existe la ruta crearla:

mkdir -p ~/git/static/static/promo-test/landing/dev/nombre_de_la_landing && git archive master | tar -x -C ~/git/static/static/promo-test/landing/dev/nombre_de_la_landing

Una vez en la carpeta del nuevo proyecyo/landing ejecutar el terminal:

npm install

Esto dejará el proyecto preparado para usar gulp y todas sus dependencias.

Para empezar a maquetar ejecutar en el terminal:

gulp

Los principales comandos son:


gulp

Crea un localhost:3000 en el navegador y carga el proyecto para poder trabajar en él y refrescar los cambios cada vez que se cambian los archivos .scss o .njk


gulp build:dev

Compila el proyecto y lo deja en la carpeta /dist si url y sin minificar


gulp build:test

Compila el proyecto y lo deja en la carpeta /dist con las rutas de test y ademas lo copia al proyecto promo-test en una carpeta con su nombre dentro (españa de momento) y sin minificar


gulp build:prod

Compila el proyecto y lo deja en la carpeta /dist con las rutas de prod y ademas lo copia al proyecto promo en una carpeta con su nombre dentro (españa de momento) con todos los archivos minimificados e imagenes optimizadas

Para ver todos los comandos (tasks) ver el archivo gulpfile.js
